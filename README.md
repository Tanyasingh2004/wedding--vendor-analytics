# 💍 Wedding Vendor Analytics & Cost Prediction

A Data Analytics and Machine Learning project that analyzes wedding vendor data and predicts estimated wedding costs based on user inputs such as city, vendor category, and vendor ratings.

## 🚀 Features
- Wedding cost prediction
- Interactive analytics dashboard
- Vendor category analysis
- Monthly revenue insights
- Data visualization
- Machine Learning model

## 🛠️ Tech Stack
- Python
- Pandas
- Scikit-learn
- Streamlit
- Matplotlib
- Joblib

## ▶️ Run
```bash
pip install -r requirements.txt
streamlit run app.py
```
